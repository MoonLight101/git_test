Images

This page covers common image formats, but you may also be looking for links and useful extars.

JPGs

JPG images are good for photos. JPGs don't allow for transparent pixels.

GIFs

GIFs are good for animations.

PNGs

PNGs are good for diagrams and icons. They do deal with opacity just fine, and they don't have color palette limitations.

SVGs

SVGs are amazing. Use them whenever you can. SVG is a vector-based graphics format, meaning it can scale up or down to any dimension without loss of quality.

But their file size may be an issue.

Dimensions

Setting only one of the attributes (height or width) will cause the image to scale proportionally, while defining both will stretch the image.

The width and height attributes can be useful, but it's usually better to set image dimensions with CSS so you can alter them with media queries.

Text alternatives

Adding alt attributes to your elements is a best practice.
It defines a “text alternative” to the image being displayed.
This has an impact on both search engines and users with text-only browsers (e.g., people that use text-to-speech software due to a vision impairment).
